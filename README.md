# Mobile App Wireframing

## Project Overview

This project focuses on designing low-fidelity wireframes for a mobile application using Canva.

## Objectives

* Conduct user research
* Create user personas
* Design user flows
* Develop low-fidelity wireframes
* Apply design thinking principles

## Tools Used

* Canva
* GitHub

## Project Deliverables

* Login Screen Wireframe
* Home Screen Wireframe
* Add Task Screen Wireframe
* User Persona
* User Flow Diagram

## Learning Outcome

This project helped me understand the fundamentals of UI/UX design, user-centered design, wireframing, and design thinking.
